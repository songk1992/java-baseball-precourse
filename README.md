## Git Command Note

```
git clone https://github.com/{본인_아이디}/{저장소 아이디}.git
git checkout -b 브랜치이름
git status // 변경된 파일 확인
git add -A(또는 .) // 변경된 전체 파일을 한번에 반영
git commit -m "메시지" // 작업한 내용을 메시지에 기록
git push origin 브랜치이름
```
